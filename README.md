# 集成战略收藏品档案

一个基于 React、TypeScript 与 Vite 的本地收藏品图鉴。

## 页面

- `/`：档案总览
- `/界园`：岁的界园志异 · 珍玩集册
- `/萨卡兹`：萨卡兹的无终奇语 · 想象实体图鉴

## 开发

```bash
npm install
npm run dev
```

质量检查：

```bash
npm run lint
npm run build
```

## 数据目录

- 萨卡兹：`public/rogue_4/collectibles.json`、`public/rogue_4/images/`
- 界园：`public/rogue_5/collectibles.json`、`public/rogue_5/images/`
